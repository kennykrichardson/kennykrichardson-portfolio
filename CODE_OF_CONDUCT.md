# 🤝 Code of Conduct

Although this repository is primarily a personal portfolio, respectful and professional interactions are expected.

## Be Respectful

- Treat everyone with respect.
- Be constructive when providing feedback.
- Keep discussions professional.

## Unacceptable Behavior

- Harassment
- Hate speech
- Personal attacks
- Spam
- Malicious contributions

## Contributions

This repository is not currently accepting external code contributions.

Issues regarding bugs, accessibility, or security are welcome.

## Enforcement

The project owner reserves the right to remove comments, issues, or discussions that violate these guidelines.

Thank you for helping maintain a positive environment.
